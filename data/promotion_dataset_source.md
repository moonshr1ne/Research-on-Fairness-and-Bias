# Employee Promotion Dataset Source

Optional second dataset:

- Kaggle: HR Analytics: Employee Promotion Data / HR Analysis Case Study
- URL: https://www.kaggle.com/datasets/shivan118/hranalysis
- License shown by Kaggle: CC0 Public Domain

Expected training file name in this repository:

`data/promotion_train.csv`

Expected columns:

- `employee_id`
- `department`
- `region`
- `education`
- `gender`
- `recruitment_channel`
- `no_of_trainings`
- `age`
- `previous_year_rating`
- `length_of_service`
- `KPIs_met >80%`
- `awards_won?`
- `avg_training_score`
- `is_promoted`

This dataset is not replaced by synthetic data. The promotion pipeline runs only when the real CSV is present.
